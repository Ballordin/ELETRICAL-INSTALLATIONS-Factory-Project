# ⚡ Industrial Electrical Installation Project
**Power Systems Design | Industrial Infrastructure | RTIEBT Compliance**

## 📖 Project Overview
This project consists of the complete electrical infrastructure design for an industrial facility (Carpentry/Machining). It covers everything from the private transformer station (PT) connection to the final circuit distribution, ensuring full compliance with Portuguese electrical safety standards (**RTIEBT**).

## 🛠️ Technical Scope
* **Power Demand:** Designed for a total installed power of **400 kVA** (contracted at **320 kVA**).
* **Voltage Levels:** 15kV (Primary) / 400V (Secondary) via a dedicated Transformer Station.
* **Calculations:** * Detailed **Power Balance** per sector (Production, Planning, Social areas).
    * **Short-Circuit Analysis** ($I_{cc}$ max/min) at the main board and partial boards.
    * **Voltage Drop** ($\Delta U$) validation for all cable runs.
* **Lighting Design:** Functional and Emergency lighting distribution.



## 🏗️ Design & Documentation
* **CAD Layouts:** Detailed AutoCAD drawings including electrical board designs, equipment distribution, and cable routing.
* **Regulatory Compliance:** Includes the official "Termo de Responsabilidade" and "Ficha Eletrotécnica" required for Portuguese licensing.
